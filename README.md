# plasma-pitch

This project is a 6.78MHz (ISM band) plasma tweeter. It uses a STM32F334 microcontroller 
for completely digital control of the modulator and RF generator. 

# Hardware
Schematics/PCB files in kicad format are in the PCB folder. 

# Sofware
Compiled with and has a makefile for GCC

For the project to build you need to standard peripheral library and 
CMSIS library from ST have to added to the lib folder.
